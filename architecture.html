📡 三、API接口清单

接口路径 方法 功能 请求参数
/api/generate-image-batch POST 多风格图片生成 { prompts: string[] }
/api/polish-text POST 语言润色 { text, context }
/api/generate-keywords POST 动态关键词生成 { keyword }
/api/analyze-mood POST 心理状态分析 { imageUrl }
/api/adjust-image POST 图片调整 { imageUrl, adjustment }
/api/generate-video POST 视频生成 { imageUrls }
/api/dreams GET POST 梦境列表/创建 —
/api/auth/login POST 用户登录 { username, password }

🧩 四、核心模块说明

🖼️ 多风格图片生成 核心

  • • 调用 batchGenerate 并行生成
  • • 4种不同风格变体
  • • 晨曦/黄昏/夜晚/暮色光线
  • • 近景/中景/远景视角
  • • 宁静/神秘/温馨/魔幻氛围

💧 动态关键词系统 创新

  • • 输入时显示河流动画
  • • 停顿2秒自动刷新
  • • AI生成相关关键词
  • • 关键词点击添加到描述
  • • 润色功能(✨按钮)

🔗 上下文关联 创新

  • • 记录上次选择的风格
  • • 场景元素延续
  • • 光线氛围延续
  • • 下次生成自动关联
  • • 例:河流场景→河边洗衣服

🧠 心理状态分析 辅助

  • • 分析图片色彩/构图
  • • 评估情绪指数(1-10)
  • • 评估压力指数(1-10)
  • • 提取心理关键词
  • • 提供正面引导建议

🖥️ 五、人机界面布局

梦境创作页面布局示意 - 用户在左侧输入描述，系统在右侧生成4种不同风格的图片供选择
