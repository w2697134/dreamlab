## 五、模块接口规范

### 5.1 API 接口清单

| 接口路径 | 方法 | 功能 | 请求参数 | 返回格式 |
|---------|------|------|---------|---------|
| `/api/generate-image-batch` | POST | 多风格图片生成 | `{ prompts: string[] }` | `{ success, results: [{imageUrl}] }` |
| `/api/generate-image` | POST | 单张图片生成 | `{ prompt, size }` | `{ success, imageUrls }` |
| `/api/polish-text` | POST | 语言润色 | `{ text, context }` | `{ success, polished }` |
| `/api/generate-keywords` | POST | 关键词生成 | `{ keyword }` | `{ success, keywords: [] }` |
| `/api/analyze-mood` | POST | 心理状态分析 | `{ imageUrl }` | `{ success, analysis: {} }` |
| `/api/adjust-image` | POST | 图片调整 | `{ imageUrl, adjustment }` | `{ success, imageUrl }` |
| `/api/generate-video` | POST | 视频生成 | `{ imageUrls: [] }` | `{ success, videoUrl }` |
| `/api/dreams` | GET/POST | 梦境列表/创建 | `?type=&page=` | `{ success, dreams: [] }` |
| `/api/dreams/[id]` | GET/PUT/DELETE | 梦境详情/修改 | `{ id, data }` | `{ success, dream }` |
| `/api/auth/login` | POST | 用户登录 | `{ username, password }` | `{ success, user }` |
| `/api/auth/register` | POST | 用户注册 | `{ username, password }` | `{ success, user }` |

### 5.2 核心数据结构

```typescript
// 已选中的图片
interface SelectedImage {
  id: string;
  prompt: string;              // 原始描述
  imageUrl: string;            // 图片URL
  timestamp: Date;              // 创建时间
  // 风格信息（用于上下文关联）
  sceneElements: string[];      // 场景元素
  lightMood: string;           // 光线氛围
  perspective: string;          // 视角
  atmosphere: string;           // 整体氛围
}

// 生成的候选图片
interface GeneratedImage {
  imageUrl: string;
  selected: boolean;
  sceneElements: string[];
  lightMood: string;
  perspective: string;
  atmosphere: string;
}

// 梦境记录
interface Dream {
  id: string;
  userId: string;
  title: string;
  type: 'image' | 'video';
  content: {
    images?: string[];
    video?: string;
    prompts?: string[];
  };
  createdAt: Date;
  updatedAt: Date;
}

// 心理状态分析结果
interface MoodAnalysis {
  overall: string;             // 整体氛围
  colorTendency: string;       // 色彩倾向
  moodLevel: number;           // 情绪指数 (1-10)
  stressLevel: number;         // 压力指数 (1-10)
  keywords: string[];          // 心理关键词
  suggestions: string;        // 建议
}
```

## 九、技术选型说明

| 层级 | 技术选型 | 理由 |
|------|---------|------|
| 框架 | Next.js 16 | App Router, SSR/SSG, API Routes |
| UI 库 | shadcn/ui | 基于 Radix UI, 可定制, 无版权 |
| 样式 | Tailwind CSS 4 | 原子化 CSS, 开发效率高 |
| AI SDK | coze-coding-dev-sdk | 统一封装多种 AI 能力 |
| 数据库 | Supabase | PostgreSQL + Auth + Storage |
| 主题 | CSS + JS | 支持明暗切换, 状态持久化 |
