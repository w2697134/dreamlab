1.1 项目背景

梦境是人类潜意识活动的窗口，记录和分析梦境有助于自我认知和心理健康。然而，梦境往往是模糊、片段化的，难以用语言准确描述。传统的梦境记录方式（如日记）缺乏直观性，无法有效捕捉梦境的视觉体验。

本项目旨在通过AI技术，将用户的文字梦境描述转化为直观的图像和视频，提供一种全新的梦境记录与探索方式，帮助用户更好地理解和分析自己的潜意识。

1.2 主要功能

  • 梦境描述输入：支持自然语言描述梦境，提供智能关键词联想
  • AI图像生成：基于Stable Diffusion生成第一人称视角的梦境图像
  • 视频串联：将多张梦境图像合成为连贯的短视频
  • 心理评估：基于梦境内容进行简单的心理状态分析
  • 梦境库管理：收藏、保存、查看历史梦境记录

1.3 性能指标

  • 图像生成时间：本地SD环境下约10-30秒
  • 关键词联想响应：< 3秒
  • 支持多端访问，响应式布局
  • 数据本地存储，保护用户隐私

第五章 安装及使用

【填写说明：简要说明安装环境要求、安装过程、主要流程等，建议包含默认安装和典型使用流程。】

5.1 安装环境要求

  • Node.js：v18.0.0 或更高版本
  • Stable Diffusion WebUI：秋葉整合包 v4.11.1 或兼容版本
  • cpolar：用于内网穿透，暴露本地SD到公网
  • 浏览器：Chrome 120+ / Firefox 121+ / Safari 16+ / Edge 120+

5.2 安装过程

5.2.1 默认安装

  1. 克隆项目代码到本地
  2. 执行 pnpm install 安装依赖
  3. 启动本地SD WebUI，确保API模式开启
  4. 配置cpolar隧道，获取公网地址
  5. 在SD配置中填入公网地址
  6. 执行 pnpm dev 启动项目
  7. 浏览器访问 http://localhost:5000

5.2.2 云端部署

  1. 将代码推送到GitHub
  2. 连接Vercel进行部署
  3. 配置环境变量（AI服务等）
  4. 自动构建并上线

5.3 典型使用流程

5.3.1 首次使用

  1. 打开首页，点击"登录"或"注册"
  2. 完成账号创建/登录
  3. 点击云朵进入创作页面

5.3.2 创建梦境

  1. 选择梦境类型（美梦/噩梦/奇幻等）
  2. 选择艺术风格（写实/水彩/油画/动漫）
  3. 输入梦境描述或点击推荐标签
  4. 点击"生成图片"
  5. 选择喜欢的图片添加到梦境集
  6. 添加2张以上可生成视频
  7. 点击"完成创作"保存

6.1 项目协调

本项目采用前后端分离架构，前端使用Next.js构建，后端借助云服务实现认证和数据管理。团队成员分工明确：一人负责前端界面和交互，一人负责AI服务集成和API开发。

6.2 克服的困难

  • 成本控制：通过使用本地SD替代云端GPU，节省了大量生成成本。
  • 网络穿透：使用cpolar解决内网穿透问题，使本地SD可被公网访问。
  • 第一人称视角：通过提示词工程优化，确保生成图像的主观沉浸感。

6.3 水平提升

  • 掌握了Next.js全栈开发能力
  • 深入学习了AI大模型的提示词工程
  • 积累了Stable Diffusion自定义模型的经验
  • 提升了响应式UI设计和用户体验优化能力

6.4 升级演进

  • 接入更多AI模型，支持更多艺术风格
  • 增加梦境分享社区功能
  • 引入更专业的心理分析算法
  • 开发小程序版本，降低使用门槛
